## 新增功能

### Win11 Fluent UI 视觉设计
- **现代化界面** - 全面升级界面美学，采用 Windows 11 风格的圆角、阴影和层级感设计
- **精细交互** - 增加了悬停动画、按下效果以及平滑过渡
- **卡片式布局** - 重定义了设置界面的卡片式布局，增加内边距和间距
- **自定义滚动条** - 实现了极简现代化的 slim 滚动条
- **圆角控件** - 对输入框、按钮、卡片等所有交互组件进行了微圆角处理 (8px/12px)

## 修复

- 修复了设置界面各部分高度不均的问题
- 优化了主界面在不同缩放下的布局表现

## 下载

下载 **Easy-Proxifier-Toggler.exe** 直接运行即可。

## 系统要求

- Windows 10/11
- 需要管理员权限

## 快速开始

1. 运行程序，点击 UAC 对话框的"是"授予权限
2. 右键托盘图标 → "设置" → 配置 Proxifier 路径
3. 点击"切换 Proxifier"一键开关
