## 新增功能

### 开机自动启动
- **自启动开关** - 在设置界面可开启/关闭 Windows 开机自启动
- **启动模式设置** - 可配置启动时是最小化到托盘还是直接打开设置界面
- **版本号显示** - 设置界面底部和托盘图标提示文字中显示版本号
- **窗口居中** - 设置窗口始终在屏幕中央显示
- **便捷启动脚本** - 新增 `start_admin.bat`，自动请求管理员权限并启动程序

## 优化改进

### 权限管理简化
- **自动 UAC 提权** - 移除复杂的交互式权限选择，改为自动请求 UAC 提权
- **开发模式** - 支持 `--dev` 参数跳过权限检查，便于调试

### UI 优化
- 调整了设置窗口布局，增加了分隔线和分组
- 完成工程化目录重构，分为 `src/core`, `src/gui`, `src/config`, `src/utils`

## 修复

- 修复了设置窗口可能会在屏幕角落显示的偏移问题
- 修复了配置保存时可能无法同步开机启动状态的问题

## 下载

下载 **Easy-Proxifier-Toggler.exe** 直接运行即可。

## 系统要求

- Windows 10/11
- 需要管理员权限

## 快速开始

1. 运行程序，点击 UAC 对话框的"是"授予权限
2. 右键托盘图标 → "设置" → 配置 Proxifier 路径
3. 点击"切换 Proxifier"一键开关
