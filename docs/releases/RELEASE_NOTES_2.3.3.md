## 新增功能

### 智能托盘图标系统
- **动态状态指示** - 托盘图标根据 Proxifier 服务状态实时切换
  - 🔵 蓝色图标 - 服务运行中
  - ⚪ 灰色图标 - 服务已停止
- **资源优化** - 按需更新机制，无后台轮询，节省系统资源
- **图标缓存** - 预加载并缓存图标文件，提升响应速度

### Win11 Fluent UI 设计
- **现代化视觉** - Windows 11 风格的圆角、阴影和层级感设计
- **卡片式布局** - 功能模块按卡片组织，层次分明
- **交互动画** - 平滑的悬停渐变效果和按钮点击动画
- **自定义滚动条** - 6px 宽度的极简滚动条
- **圆角控件** - 输入框、按钮、卡片等组件微圆角处理 (8px/12px)

### 模块化组件架构
- **状态监控面板** - 实时显示服务和进程状态，异步监控（每 2 秒刷新）
- **配置管理面板** - Proxifier 路径配置，文件浏览器集成
- **启动设置面板** - 开机自启动和启动最小化选项
- **关于对话框** - 版本信息展示，GitHub 链接跳转

## 技术改进

- 线程安全架构 - Toplevel 窗口架构，避免线程冲突
- 配置管理系统 - JSON 格式配置文件，热更新支持
- 权限管理 - 自动 UAC 提权，开发模式支持

## 下载

下载 **Easy-Proxifier-Toggler.exe** 直接运行即可。

## 系统要求

- Windows 10/11
- 需要管理员权限
- Python 3.8+ (源码运行)

## 快速开始

1. 运行程序，点击 UAC 对话框的"是"授予权限
2. 右键托盘图标 → "主界面" → 配置 Proxifier 路径
3. 点击"切换 Proxifier"一键开关
