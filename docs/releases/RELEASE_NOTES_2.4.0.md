## 新增功能

### 托盘菜单快捷开关
- **开机自启动开关** - 直接在托盘菜单切换开机自启动，无需打开设置界面
- **最小化启动开关** - 控制程序启动时是否最小化到托盘
- **打勾状态指示** - 启用时菜单项前显示 ✓ 标记，状态一目了然
- **即时通知反馈** - 切换后立即显示系统通知，操作结果清晰可见

### 托盘菜单结构优化
```
托盘菜单
├── 切换 Proxifier (默认双击操作)
├── 查看状态
├── ─────────────
├── 主界面
├── ─────────────
├── ✓ 开机自启动
├── ✓ 最小化启动
├── ─────────────
└── 退出
```

## UI/UX 优化

### 状态面板布局重构
- **单行紧凑布局** - 驱动服务和进程状态并排显示，节省空间
- **等宽徽章设计** - 状态徽章统一宽度（120px），视觉更协调
- **固定面板高度** - 优化为 180px，减少不必要的留白
- **竖线分隔符** - 使用 | 分隔两个状态区域，层次更清晰
- **精细间距调整** - 优化各元素间距，布局更紧凑美观

## 技术改进

- 配置实时同步 - 托盘菜单开关状态自动保存到配置文件
- 注册表自动管理 - 开机自启动自动同步 Windows 注册表
- 容器尺寸控制 - 精确控制布局
- 按钮圆角统一 - 切换按钮圆角统一为 8px

## 下载

下载 **Easy-Proxifier-Toggler.exe** 直接运行即可。

## 系统要求

- Windows 10/11
- 需要管理员权限

## 快速开始

1. 运行程序，点击 UAC 对话框的"是"授予权限
2. 右键托盘图标 → "主界面" → 配置 Proxifier 路径
3. 点击"切换 Proxifier"一键开关
